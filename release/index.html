使用说明

方式一: 使用本地服务器

运行 python -m http.server 8080 或 npx serve .

然后在浏览器中打开 http://localhost:8080

方式二: 直接打开

双击 index.html 直接在浏览器中打开（部分功能可能受限）

方式三: 生成后使用

生成的 PPT 文件在 output 目录中，可以直接打开 index.html 预览
